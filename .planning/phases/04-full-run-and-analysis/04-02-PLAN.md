Sacred metrics.json format:
```json
{
  "normalized_score": {
    "values": [0.1, 0.3, 0.5],
    "steps": [0, 1, 2],
    "timestamps": ["..."]
  }
}
```

Sacred config.json contains: total_timesteps (int), n_rounds (int)
BC has only step=0 entry. DAgger/FTRL have steps 0..N-1.
Round-to-timestep conversion: timestep = round_num * (total_timesteps / n_rounds)
