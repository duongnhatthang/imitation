Create `experiments/run_atari_experiment.py` — a Sacred-instrumented entry point that runs ONE (algorithm, game, seed) combination and writes to an isolated FileStorageObserver directory.

The script must:

1. **Pre-parse CLI args** using argparse with `parse_known_args()` BEFORE Sacred consumes argv. Extract `--algo`, `--game`, `--seed`, `--output-dir` to build the observer path. This is necessary because Sacred's `run_commandline()` consumes sys.argv and we need the observer path before that.

2. **Sacred experiment config** with these defaults:
   - `algo = "dagger"` (one of: bc, dagger, ftrl)
   - `game = "Pong"` (one of the 7 ATARI_GAMES keys)
   - `seed = 0`
   - `n_rounds = 20`
   - `total_timesteps = 500000`
   - `n_envs = 8`
   - `alpha = 1.0` (FTRL only)
   - `output_dir = "output/sacred"`

3. **FileStorageObserver path**: `{output_dir}/{algo}/{game}/{seed}` — this ensures each (algo, game, seed) combination has its own Sacred directory, eliminating run-ID collisions (INFRA-04).

4. **Main function** that:
   - Imports from `experiments.atari_smoke`: `run_bc`, `run_dagger`, `load_random_baselines`
   - Imports `ATARI_GAMES` from `experiments.atari_helpers`
   - Loads random baselines via `load_random_baselines()`
   - Gets `random_score = baselines[game]["mean"]`
   - Creates `rng = np.random.default_rng(seed)`
   - For `algo == "bc"`: calls `run_bc(game, game_id, seed, n_envs, random_score, rng)`
   - For `algo in ("dagger", "ftrl")`: calls `run_dagger(game, game_id, seed, n_envs, n_rounds, total_timesteps, random_score, rng, use_ftrl=(algo=="ftrl"), alpha=alpha)`
   - Logs `normalized_score` via `_run.log_scalar("normalized_score", norm_score)`
   - Logs final round count warning if DAgger/FTRL produced fewer than `n_rounds` rounds
   - Returns `{"normalized_score": norm_score}`

5. **sys.path setup** at top of file: `_PROJECT_ROOT = Path(__file__).resolve().parents[1]` and `sys.path.insert(0, str(_PROJECT_ROOT))` — same pattern as `atari_smoke.py`.

6. **`if __name__ == "__main__"` block** that:
   - Pre-parses `--algo`, `--game`, `--seed`, `--output-dir` with `parse_known_args()`
   - Appends `FileStorageObserver(obs_path)` to `ex.observers`
   - Calls `ex.run_commandline()`

The script must be runnable as:
```bash
python experiments/run_atari_experiment.py with algo=dagger game=Pong seed=0 n_rounds=20 total_timesteps=500000
```

Note: Sacred uses `with key=value` syntax on command line, NOT `--key value`. The pre-parser uses `--algo` etc. for the observer path construction, but Sacred config overrides use `with algo=X`.
