**tmux auto-relaunch (INFRA-02):**
If the script is not already inside a tmux session (`[ -z "${TMUX:-}" ]`), re-launch itself inside a named tmux session `atari_bench_${TIMESTAMP}`. Print the session name and attach command, then exit. The inner invocation gets a `--_in-tmux` flag to skip the re-launch check.

```bash
if [[ "${1:-}" != "--_in-tmux" ]] && [ -z "${TMUX:-}" ]; then
  SESSION="atari_bench_$(date +%Y%m%d_%H%M%S)"
  tmux new-session -d -s "$SESSION" "bash $0 --_in-tmux; echo 'Benchmark complete. Press enter to close.'; read"
  echo "Launched in tmux session '$SESSION'"
  echo "Attach with: tmux attach -t $SESSION"
  echo "Detach with: Ctrl-b d"
  exit 0
fi
# Shift off the --_in-tmux flag if present
[[ "${1:-}" == "--_in-tmux" ]] && shift
```

**Job summary:**
Print the number of combinations (${#ALGOS[@]} * ${#GAMES[@]} * ${#SEEDS[@]}), output directory, and GPU count before launching.

**GNU parallel invocation (INFRA-02):**
```bash
parallel --jobs ${N_GPUS} --eta --halt soon,fail=1 --joblog "${OUTPUT_DIR}/joblog.txt" \
  "CUDA_VISIBLE_DEVICES=\$(( {%} - 1 )) python experiments/run_atari_experiment.py \
    --algo {1} --game {2} --seed {3} --output-dir ${OUTPUT_DIR} \
    with algo={1} game={2} seed={3} \
    n_rounds=${N_ROUNDS} total_timesteps=${TOTAL_TIMESTEPS} \
    n_envs=${N_ENVS} alpha=${ALPHA}" \
  ::: "${ALGOS[@]}" \
  ::: "${GAMES[@]}" \
  ::: "${SEEDS[@]}"
```

Key details:
- `--jobs ${N_GPUS}` limits concurrency to 4 (one per GPU)
- `--eta` shows estimated time of arrival
- `--halt soon,fail=1` stops scheduling new jobs on first failure but lets running jobs finish
- `--joblog` writes a per-job log for debugging
- `CUDA_VISIBLE_DEVICES=$(( {%} - 1 ))` maps 1-indexed slot to 0-indexed GPU
- Both `--algo {1}` (for pre-parser) and `with algo={1}` (for Sacred config) are passed
- `:::` produces the cartesian product of ALGOS x GAMES x SEEDS
