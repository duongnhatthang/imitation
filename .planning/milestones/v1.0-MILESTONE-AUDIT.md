### EVAL-02 Detail

**Requirement:** Learning curves plotted: normalized return vs environment interactions for each game

**Issue:** `run_atari_experiment.py` logs only the final `normalized_score` to Sacred (one `_run.log_scalar` call). `plot_learning_curves` expects a per-round time series. The fallback path (constant interpolation) produces valid PDFs but DAgger/FTRL appear as flat horizontal lines — scientifically degenerate.

**Fix:** Add per-round evaluation callback to `run_dagger`/`run_ftrl` in `atari_smoke.py` that calls `_run.log_scalar("normalized_score", score, step=round_idx)` after each DAgger round's evaluation rollout.
