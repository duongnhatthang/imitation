**Class 1: FTRLLossCalculator**
- `@dataclasses.dataclass(frozen=True)` to match BehaviorCloningLossCalculator pattern
- Fields:
  - `bc_loss_calculator: BehaviorCloningLossCalculator` -- the base BC loss calculator
  - `anchor_params: List[th.Tensor]` -- frozen snapshot of w_t (detached, not in graph)
  - `sigma_grad: List[th.Tensor]` -- frozen sum of past-loss gradients at w_t (detached)
  - `eta_t: float` -- FTRL learning rate for this round
- `__call__(self, policy, obs, acts) -> BCTrainingMetrics`:
  1. Call `self.bc_loss_calculator(policy, obs, acts)` to get base `bc_metrics`
  2. Compute proximal term: `sum(th.sum((p - a) ** 2) for p, a in zip(policy.parameters(), self.anchor_params)) / (2.0 * self.eta_t)`
     - NOTE: `a` is already detached (stored detached), but use `a.detach()` defensively
  3. Compute linear correction: `-sum(th.sum(p * g.detach()) for p, g in zip(policy.parameters(), self.sigma_grad))`
     - The negative sign is critical (per Eq. 6)
  4. Compute `total_loss = bc_metrics.loss + proximal_term + linear_correction`
  5. Return a NEW `BCTrainingMetrics` with all fields from `bc_metrics` EXCEPT `loss` replaced by `total_loss`:
     ```python
     return dataclasses.replace(bc_metrics, loss=total_loss)
     ```
     (Use `dataclasses.replace` since BCTrainingMetrics is frozen)

**Class 2: FTRLDAggerTrainer**
- Subclass of `dagger.SimpleDAggerTrainer`
- Constructor `__init__(self, *, alpha: float = 1.0, **kwargs)`:
  - Call `super().__init__(**kwargs)`
  - Store `self.alpha = alpha`
  - Initialize `self._cumulative_sigma = 0.0` (accumulated sum of sigma_i values)
  - Store `self._default_bc_loss_calculator = self.bc_trainer.loss_calculator` (save the original BC loss calculator for use inside FTRLLossCalculator)
- Override `extend_and_update(self, bc_train_kwargs=None) -> int`:

     ```python
     if bc_train_kwargs is None:
     ```

  2. Call `self._try_load_demos()` to populate the data loader
  3. Snapshot anchor: `anchor_params = [p.detach().clone() for p in self.bc_trainer.policy.parameters()]`
  4. Compute sigma_grad:
     - If `self.round_num == 0`: `sigma_grad = [th.zeros_like(p) for p in self.bc_trainer.policy.parameters()]`
     - Else: accumulate gradients over the full dataset:
       ```python
       self.bc_trainer.policy.zero_grad()
       base_calc = self._default_bc_loss_calculator
       for batch in self.bc_trainer._demo_data_loader:
           obs_tensor = types.map_maybe_dict(
               lambda x: util.safe_to_tensor(x, device=self.bc_trainer.policy.device),

           )
           acts = util.safe_to_tensor(batch["acts"], device=self.bc_trainer.policy.device)
           metrics = base_calc(self.bc_trainer.policy, obs_tensor, acts)
           metrics.loss.backward()
       sigma_grad = [p.grad.detach().clone() if p.grad is not None else th.zeros_like(p)
                     for p in self.bc_trainer.policy.parameters()]
       self.bc_trainer.policy.zero_grad()
       ```
       Import `util` from `stable_baselines3.common` (it's `from stable_baselines3.common.utils import safe_to_tensor` -- check bc.py imports for the exact pattern; it uses `from torch.utils.data import default_collate` or similar.   5. Compute eta_t:
     - Use `sigma_i = 1.0` (constant per-round, simplest approach per research recommendation)
     - `self._cumulative_sigma += sigma_i`
     - `eta_t = self.alpha / self._cumulative_sigma`
     - NOTE: parameterize as `eta_t = alpha / (cumulative_sigma)` so that large alpha = large eta = weak proximal = FTL degeneracy (this makes ALGO-05 testable: alpha -> infinity means eta -> infinity means proximal coeff 1/(2*eta) -> 0)
  6. Inject FTRLLossCalculator:

     ```python
         anchor_params=anchor_params,
         sigma_grad=sigma_grad,
         eta_t=eta_t,
     )
     ```
  7. Train and advance round:
     ```python

     self.bc_trainer.train(**bc_train_kwargs)
     self.round_num += 1
     logging.info(f"FTRL new round number is {self.round_num}")
     return self.round_num
     ```

**Critical implementation notes (from research pitfalls):**
- Anchor params MUST be `detach().clone()` -- no graph connection
- sigma_grad MUST be `detach().clone()` after backward -- no graph connection
- `policy.zero_grad()` MUST be called AFTER extracting sigma_grad to prevent stale gradients leaking into BC.train()
- Round 0 sigma_grad is all zeros (no past data to compute gradients over)
- `_try_load_demos()` MUST be called BEFORE sigma_grad computation so the data loader is populated
- The data loader iteration for sigma_grad does NOT call optimizer.step() -- it only accumulates .grad
- A NEW FTRLLossCalculator is created each round (anchor and sigma_grad change each round)
